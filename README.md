## Project Description
TaskFlow is a lightweight, browser-based task management application built with Angular v21. It allows users to track tasks with specific priorities, due dates, and statuses using a clean Bootstrap 5 interface.

## Core Features
- **CRUD Operations:** Add, Edit, View, and Delete tasks.
- **Nested Routing:** Task details include sub-views for Info and Editing.
- **Responsive Design:** Fully functional on mobile and desktop via Bootstrap 5.
- **Real-time Updates:** Status toggling reflects immediately in the task list.

## Tech Stack
- **Framework:** Angular v21 (Standalone Components)
- **Language:** TypeScript
- **Styling:** Bootstrap 5
- **Icons:** Bootstrap Icons

## How to Run Locally
1. Clone the repo: `git clone <your-repo-link>`
2. Install dependencies: `npm install`
3. Run the app: `ng serve`
4. Open `http://localhost:4200`
